# Vue + Vuetify boilerplate

A skeleton for a new Vue web application project.

Vue stack:
- Vue (2.6.12)
- Vuetify (2.3.15)
- Vuex (3.5.1)
- Vue Router (3.4.7)

Maps:
- Leaflet
- Vue2Leaflet

Tests:
- Mocha
- Chai

Other:
- Google Analytics
- Vuetify Confirm
- configured sass styles
- local storage management service

### How to run?
1. Install dependencies:
`npm install`

2. Run server:
`npm run serve`

3. Build:
`npm run build`

4. Run linter:
`npm run lint`

5. Run tests:
`npm run test`
